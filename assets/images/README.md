# Images

Place your couple photo here as `hero.jpg` (or any format).

**To use a real photo**, replace the `.couple-photo-placeholder` `<div>` in `index.html` with:

```html
<img
  src="assets/images/hero.jpg"
  alt="Foto Raka dan Sari"
  class="hero-photo fade-in"
/>
```

**Recommended dimensions:** 600×600 px (square, will be displayed as circle)
