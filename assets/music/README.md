# Background Music

Place your background music file here as `background.mp3`.

**Requirements:**
- Format: MP3 (best compatibility)
- Recommended bitrate: 128 kbps (balance of quality and file size)
- Recommended length: 3–5 minutes (it loops automatically)

**To change the file path**, update `CONFIG.musicSrc` in `assets/js/script.js`.

> ⚠️ Music does NOT autoplay. The user must click the ▶ button (bottom-right corner) to start.
